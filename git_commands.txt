✅ Step-by-Step Guide to Pull GitHub Updates
🧭 First: Open Terminal or Command Prompt
Go to your project folder in the terminal or VS Code terminal.

🔁 1. Check current branch:

git branch
You’ll see something like:

* main
If you're not on the correct branch (main or master), switch to it:


git checkout main

🔄 2. Pull latest changes from GitHub:

git pull origin main

Replace main with your branch name if different.

🧹 3. (Optional) If your files have conflicts or you're unsure:

git stash          # Save local changes temporarily
git pull origin main
git stash pop      # Reapply your changes after update

'===================================================================================='

🔁 Updating Existing GitHub Repository
If you’ve already pushed before, and just want to update:

1. Stage your changes

git add .
2. Commit changes

git commit -m "Updated feature X or fixed bug Y"
3. Push

git commit -m "Updated feature X or fixed bug Y"

أنواع التقارير والتحليلات المالية الممكنة
1. تحليل أوامر الشراء (Purchase Orders Analysis)

إجمالي حجم المشتريات: قيمة إجمالية لكل فترة زمنية (يومي، أسبوعي، شهري).

عدد الأوامر: مراجعة عدد الطلبات المفتوحة مقابل المكتملة.

متوسط قيمة الطلب: حساب المتوسط لمعرفة الاتجاهات.

توزيع الموردين: أعلى الموردين من حيث عدد الطلبات أو القيمة المالية.

2. تحليل المخزون (Inventory Analysis)

تكلفة المخزون الحالية: كمية × سعر، لكل مادة.

معدلات دوران المخزون: مقارنة بين الكميات الداخلة والخارجة.

مواد بطيئة الحركة (Slow-moving): تحديد العناصر المخزنية التي لم تتحرك منذ فترة طويلة.

مواد نفد منها المخزون (Out-of-stock Alerts): مراقبة المنتجات التي تحتاج لإعادة طلب.

3. تحليل الموردين (Suppliers Analysis)

التكاليف حسب المورد: إجمالي الإنفاق لكل مورد.

📊 أنواع التقارير والتحليلات المالية الممكنة
1. 📝 تحليل أوامر الشراء (Purchase Orders Analysis)

إجمالي حجم المشتريات

تجميع قيمة جميع أوامر الشراء حسب الفترة (يومي / أسبوعي / شهري).

عدد الأوامر

مقارنة بين الأوامر "المكتملة" و"المفتوحة" و"الملغية".

متوسط قيمة الطلب

(إجمالي قيمة المشتريات ÷ عدد الأوامر).

توزيع الموردين

تصنيف الموردين بالأكثر تعاملًا من حيث العدد أو إجمالي القيمة.

2. 📦 تحليل المخزون (Inventory Analysis)

تكلفة المخزون الحالية

(الكمية × سعر الوحدة) لكل صنف، مع الإجمالي الكلي.

معدلات دوران المخزون (Inventory Turnover)

(إجمالي المخزون المباع ÷ متوسط المخزون خلال الفترة).

مواد بطيئة الحركة (Slow-moving)

الأصناف اللي ما اتحركتش من المخزن لفترة زمنية معينة.

مواد نفد منها المخزون (Out-of-stock Alerts)

تنبيه عند وصول الكمية للمستوى الأدنى (Reorder Point).

3. 🏭 تحليل الموردين (Suppliers Analysis)

التكاليف حسب المورد

تجميع قيمة أوامر الشراء لكل مورد لمعرفة حجم الإنفاق عليه.

ترتيب الموردين

Top 5 Suppliers بالأكثر تكلفة.

مقارنة الموردين

نسبة كل مورد من إجمالي الإنفاق (Pie Chart).
